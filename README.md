# Random Quote Machine

A simple and interactive **Random Quote Machine** built using **React**.  
This project displays a random quote with its author and allows users to generate a new quote or share the current quote on Twitter.

## Features

- Displays a random quote on page load
- Shows the quote author
- Generates a new random quote on button click
- Tweet sharing functionality
- Responsive and clean user interface
- Centered quote box layout

## Technologies Used

- HTML
- CSS
- JavaScript
- React
- Font Awesome

## Project Requirements Covered

This project fulfills the main requirements of the freeCodeCamp **Random Quote Machine** challenge:

- `id="quote-box"`
- `id="text"`
- `id="author"`
- `id="new-quote"`
- `id="tweet-quote"`

It also includes:
- Random quote generation
- Dynamic author update
- Twitter intent sharing link
- Centered layout design

## How to Run the Project

1. Download or clone the project
2. Open the project folder in VS Code
3. Save the code in a file named `index.html`
4. Open it in the browser

You can also run it using the **Live Server** extension in VS Code for a better experience.
